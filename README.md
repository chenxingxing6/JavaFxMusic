### 此播放器实现了酷狗7的大部分功能:
    1. 歌曲分组管理里
    2. 系统托盘
    3. 分组重命名
    4. 移动歌曲
    5. 移除歌曲(非删除)
    6. 歌曲按命名排序
    7. 歌曲搜索
    8. 桌面歌词秀
